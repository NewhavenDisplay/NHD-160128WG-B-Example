# NHD-160128WG-Example
Example Arduino code for NHD-160128WG.

Target: `Arduino MEGA`\
Language: `Arduino Sketch`\
Interface: `8-bit Parallel`\
Display Voltage: `5V`\
Logic Voltage: `5V`
